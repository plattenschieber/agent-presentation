# Agents Anonymous 🕵️ Cologne Edition - Round 6

## Event Description

**A safe space for builders using agentic coding tools.** Claude Code, Cursor, Windsurf, Copilot, Pi, Aider, Cline — whatever you use, you're welcome. The patterns are the same. The stories are universal.

5 rounds in (formerly as "Claude Code Anonymous"), we're opening up. Same format, same energy, bigger tent.

**This time: Make It Yours — Customizing Your Agent Harness**

Out-of-the-box is just the starting line. We'll talk about turning generic tools into YOUR tool: config files, skills, extensions, multi-agent orchestration, and the patterns that make agent setups truly personal.

**What makes this different:**
- Nothing is recorded or live-streamed
- What's said in the room, stays in the room
- Builders with real hands-on experience
- Lightning talks from participants (5 min each)
- Open discussions about what actually works (and what doesn't)

**Format:**
- 18:00 - Arrive and get-together
- 18:45 - Opening talk: "Make It Yours" (Pi extensions, Armin Ronacher's agent-stuff, building your own cockpit)
- 19:00 - Lightning talks (5 min each, ~45 min)
- 19:45 - Wrap-up and open discussion
- 20:30 - Venue concludes

**Speakers get a Kefir.** It's a thing. Don't ask. Just present and find out.

**Who should come:**
Developers, founders, engineers using any agentic coding tool — whether you've heavily customized your setup or are curious how others do it. Tool-agnostic, experience-level-agnostic. Bring what you've got.

**Location:**
Cologne, Germany (exact address shared upon approval)
